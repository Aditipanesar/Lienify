# 🚀 Lienify - URL Shortener

A full-stack URL shortening application with authentication and analytics.

## 🔗 Live Demo
Frontend: https://reliable-bonbon-fbd250.netlify.app  
Backend: https://lienify.onrender.com  

## ⚙️ Tech Stack
- Frontend: React, Tailwind CSS
- Backend: Spring Boot
- Database: PostgreSQL
- Authentication: JWT
- Deployment: Docker, Render, Netlify

## ✨ Features
- User registration & login
- Shorten long URLs
- Redirect using short links
- Track click analytics
- Secure APIs with JWT

## 🐳 Deployment
- Backend containerized using Docker
- Deployed on Render
- Frontend deployed on Netlify
